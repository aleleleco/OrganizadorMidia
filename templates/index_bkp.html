Como utilizar

Para organização das midias em pastas pode ser utilizada buscando configurações direta do arquivo, das propriedades EXIF, ou de arquivo json (utilizado pelo google takeout)

  • Buscar EXIF - Melhor opção, faz a busca das propriedades do EXIF de cada imagem. de acordo com a data de criação organiza os arquivos
  • Buscar Propriedades do arquivo - Faz a busca das propriedades, apesar da informação existir pode não estar de acordo com a correta data de criaçao do arquivo já que utiliza a data de modificação
  • Arquivos Json - Feito para buscar quando as imagens são esportadas do Google takeout, ele gera as informaçoes em um arquivo json. Use somente para este tipo caso

Busca dados dos arquivos em Json (Exportados Google takeout)

Faz a busca da data em que foi realizada a captura da imagem em um arquivo json com os dados exif da foto, utilizado normalmente quando feita a exportaçao dos daos pelo Google Takeout

Por padrao os arquivos exportaos vem com nome_arquivo.jpg, nome_arquivo.jpg.json

Quando exportado dados do Takeout ele gera arquivos em pastas e nao sao todos arquivos de img e json que ficam na mesma pasta, para uma busca mais precisa aconselhado deixar todos os arquivos na mesma pasta, apesar do programa buscar as imagens em todoas as pastas, isto pode economizar muito tempo dependendo quantidade de imagens.

Preencha o caminho da pasta raiz onde estão todas as fotos e os arquivos .json exportados
