Como utilizar

Este software tem como objetivo orgtanizar fotos, desenvolvido para subrir necessidade pessoal de organizar as fotos por pastas levando em consideração a data de captura ou de edição

existem 3 maneira de utilizar este software

  • Arquivos exportados google takeout - ao fazer o download dos arquivos do google takeout ele exporta as informações das imagens para um arquivos json contendo o mesmo nome da imagem atravez destes arquivos é possivel buscar a data que a imagem foi modificada e assim organizar por data
  • Arquivos de imagens baixados - sem informação EXIF - Busca a data de alteraçào do properties da imagem
  • Arquivos de imagens - con informação EXIF - Busca a data em que a foto foi tirada direto na informação EXIF da foto

Busca dados dos arquivos em Json (Exportados Google takeout)

Faz a busca da data em que foi realizada a captura da imagem em um arquivo json com os dados exif da foto, utilizado normalmente quando feita a exportaçao dos daos pelo Google Takeout

Por padrao os arquivos exportaos vem com nome_arquivo.jpg, nome_arquivo.jpg.json

Quando exportado dados do Takeout ele gera arquivos em pastas e nao sao todos arquivos de img e json que ficam na mesma pasta, para uma busca mais precisa aconselhado deixar todos os arquivos na mesma pasta, apesar do programa buscar as imagens em todoas as pastas, isto pode economizar muito tempo dependendo quantidade de imagens.

Preencha o caminho da pasta raiz onde estão todas as fotos e os arquivos .json exportados

tipos de midia para buscar:

Busca dados direto nos arquivos

Faz a busca nas propriedade dos arquivos, buscando data de modificação do arquivo, menos precisa do que quando existe informaçào exif.

Preencha o caminho da pasta raiz onde estão todas as fotos

Busca dados direto nos arquivos (EXIF)

Faz a busca nas propriedade dos arquivos, especifica quando existe informação EXIF, comun em imagens de cameras digitais e celuares possivel escolher qual o tipo de arquivo ler

Preencha o caminho da pasta raiz onde estão todas as fotos
